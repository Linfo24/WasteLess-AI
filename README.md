# WasteLess AI â€” Smart Food Waste Reduction Assistant

WasteLess AI is a full-stack web application that helps users reduce food waste by generating:
- A 3-day optimized meal plan using the ingredients they already have.
- AI-generated recipe images.
- A dynamic pantry that tracks expiration dates.

The system uses **GPT-4o-mini** to generate meal plans and high-quality recipe descriptions entirely in English.

---

##  Features

###  AI-Generated Meal Plans
Users input their available ingredients and receive a structured 3-day meal plan.

###  Real-Time Image Generation  
Each recipe triggers an image generation request.

###  Smart Pantry  
Items include:
- name  
- quantity  
- unit  
- expiration date  

###  Modern Responsive UI  
Clean, React-based interface with intuitive interactions.
