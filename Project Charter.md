## 3. Problem Statement
Households waste a significant amount of food due to poor meal planning and forgotten ingredients. There is a need for a system that helps families use ingredients before they expire.

## 4. Proposed Solution & Core Features
WasteLess AI is an intelligent meal-planning assistant that:
- Generates optimized 3-day meal plans  
- Prioritizes ingredients close to expiration  
- Suggests recipes using pantry items  
- Generates realistic meal images  
- Reduces overall food waste  

## 5. Novelty Statement
WasteLess AI is not a traditional recipe app.  
Its novelty lies in:
- A custom AI prompt-engineering logic focused on waste reduction  
- A dual AI system (text + image generation)  
- Optimization based on expiration urgency  
- A dedicated sustainability-focused planning model  

## 6. Tech Stack
**Frontend:** React, Vite, TypeScript  
**Backend:** FastAPI, Python, SQLModel  
**Database:** SQLite  
**AI Models:** GPT-4o-mini, GPT-image-1
